# Data

This folder contains small, local datasets used by the notebooks.

## Included
- `energy_efficiency.csv`: used for optional regression exercises.

## Adding new datasets
If a notebook references a dataset not included in this folder, place the CSV file here and update this README with:
1. A short description of the dataset.
2. A manual download link.
3. Any preprocessing steps needed before running the notebook.

> Note: No notebooks rely on the Kaggle API or authentication.
